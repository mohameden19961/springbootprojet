## 📡 Principes REST Implémentés

### Codes HTTP Utilisés

| Code | Signification | Utilisation |
|------|---------------|------------|
| 200 | OK | GET, PUT retour successful |
| 201 | Created | POST création successful |
| 204 | No Content | DELETE successful |
| 400 | Bad Request | Données invalides |
| 404 | Not Found | Ressource non trouvée |
| 409 | Conflict | Ressource dupliquée |
| 500 | Internal Error | Erreur serveur |

### Structure des URLs

```
/api/v1/resources              # Liste paginée
/api/v1/resources/{id}         # Détail d'une ressource
/api/v1/resources              # POST pour créer
/api/v1/resources/{id}         # PUT pour modifier
/api/v1/resources/{id}         # DELETE pour supprimer
/api/v1/resources/search       # GET pour rechercher
/api/v1/resources/filter/{key} # GET pour filtrer
```

### 1. BookController
```java
@RestController
@RequestMapping("/api/v1/books")
@RequiredArgsConstructor
@Validated
public class BookController {
    
    private final BookService bookService;
    
    // ============ CRUD OPERATIONS ============
    
    @PostMapping
    public ResponseEntity<BookDTO> createBook(@Valid @RequestBody BookDTO bookDTO) {
        return ResponseEntity.status(HttpStatus.CREATED)
                .body(bookService.createBook(bookDTO));
    }
    
    @GetMapping
    public ResponseEntity<Page<BookDTO>> getAllBooks(Pageable pageable) {
        return ResponseEntity.ok(bookService.getAllBooks(pageable));
    }
    
    @GetMapping("/{id}")
    public ResponseEntity<BookDTO> getBookById(@PathVariable Long id) {
        return ResponseEntity.ok(bookService.getBookById(id));
    }
    
    @PutMapping("/{id}")
    public ResponseEntity<BookDTO> updateBook(
            @PathVariable Long id,
            @Valid @RequestBody BookDTO bookDTO) {
        return ResponseEntity.ok(bookService.updateBook(id, bookDTO));
    }
    
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> deleteBook(@PathVariable Long id) {
        bookService.deleteBook(id);
        return ResponseEntity.noContent().build();
    }
    
    // ============ SEARCH & FILTER ============
    
    // Recherche par ISBN
    @GetMapping("/isbn/{isbn}")
    public ResponseEntity<BookDTO> getBookByIsbn(@PathVariable String isbn) {
        return ResponseEntity.ok(bookService.getBookByIsbn(isbn));
    }
    
    // Recherche par titre
    @GetMapping("/search/title")
    public ResponseEntity<Page<BookDTO>> searchByTitle(
            @RequestParam String title,
            Pageable pageable) {
        return ResponseEntity.ok(bookService.searchBooksByTitle(title, pageable));
    }
    
    // Filtrer par catégorie
    @GetMapping("/filter/category/{categoryId}")
    public ResponseEntity<Page<BookDTO>> getByCategory(
            @PathVariable Long categoryId,
            Pageable pageable) {
        return ResponseEntity.ok(bookService.findBooksByCategory(categoryId, pageable));
    }
    
    // Filtrer par auteur
    @GetMapping("/filter/author/{authorId}")
    public ResponseEntity<Page<BookDTO>> getByAuthor(
            @PathVariable Long authorId,
            Pageable pageable) {
        return ResponseEntity.ok(bookService.findBooksByAuthor(authorId, pageable));
    }
    
    // Filtrer par langue
    @GetMapping("/filter/language/{languageCode}")
    public ResponseEntity<Page<BookDTO>> getByLanguage(
            @PathVariable String languageCode,
            Pageable pageable) {
        return ResponseEntity.ok(bookService.findBooksByLanguage(languageCode, pageable));
    }
    
    // ============ BOOK ITEMS ============
    
    // Ajouter un exemplaire
    @PostMapping("/{bookId}/items")
    public ResponseEntity<BookItemDTO> addBookItem(
            @PathVariable Long bookId,
            @Valid @RequestBody BookItemDTO bookItemDTO) {
        return ResponseEntity.status(HttpStatus.CREATED)
                .body(bookService.createBookItem(bookId, bookItemDTO));
    }
    
    // Lister exemplaires
    @GetMapping("/{bookId}/items")
    public ResponseEntity<Page<BookItemDTO>> getBookItems(
            @PathVariable Long bookId,
            Pageable pageable) {
        return ResponseEntity.ok(bookService.getBookItemsByBook(bookId, pageable));
    }
    
    // Détail d'un exemplaire
    @GetMapping("/items/{itemId}")
    public ResponseEntity<BookItemDTO> getBookItem(@PathVariable Long itemId) {
        return ResponseEntity.ok(bookService.getBookItemById(itemId));
    }
    
    // Compter disponibles
    @GetMapping("/{bookId}/available-count")
    public ResponseEntity<Long> getAvailableCount(@PathVariable Long bookId) {
        return ResponseEntity.ok(bookService.countAvailableItemsByBook(bookId));
    }
}
```

**Endpoints** :
- `POST /api/v1/books` : Créer livre
- `GET /api/v1/books` : Lister livres (pagined)
- `GET /api/v1/books/{id}` : Détail livre
- `GET /api/v1/books/isbn/{isbn}` : Rechercher par ISBN
- `GET /api/v1/books/search/title?title=query` : Rechercher par titre
- `GET /api/v1/books/filter/category/{id}` : Filtrer catégorie
- `GET /api/v1/books/filter/author/{id}` : Filtrer auteur
- `GET /api/v1/books/filter/language/{code}` : Filtrer langue
- `PUT /api/v1/books/{id}` : Modifier livre
- `DELETE /api/v1/books/{id}` : Supprimer livre
- `POST /api/v1/books/{bookId}/items` : Ajouter exemplaire
- `GET /api/v1/books/{bookId}/items` : Lister exemplaires
- `GET /api/v1/books/items/{itemId}` : Détail exemplaire

### 8. ReferenceController (Données de Référence)
```java
@RestController
@RequestMapping("/api/v1/reference")
@RequiredArgsConstructor
public class ReferenceController {
    
    private final LanguageRepository languageRepository;
    private final NationalityRepository nationalityRepository;
    
    // ============ LANGUAGES ============
    
    @GetMapping("/languages")
    public ResponseEntity<List<LanguageDTO>> getAllLanguages() {
        List<LanguageDTO> languages = languageRepository.findAll().stream()
                .map(l -> new LanguageDTO(l.getCode(), l.getName()))
                .collect(Collectors.toList());
        return ResponseEntity.ok(languages);
    }
    
    @GetMapping("/languages/{code}")
    public ResponseEntity<LanguageDTO> getLanguage(@PathVariable String code) {
        return languageRepository.findById(code)
                .map(l -> ResponseEntity.ok(new LanguageDTO(l.getCode(), l.getName())))
                .orElseGet(() -> ResponseEntity.notFound().build());
    }
    
    @PostMapping("/languages")
    public ResponseEntity<LanguageDTO> createLanguage(@RequestBody LanguageDTO languageDTO) {
        var language = languageRepository.save(new Language(
                languageDTO.getCode(),
                languageDTO.getName()
        ));
        return ResponseEntity.status(HttpStatus.CREATED)
                .body(new LanguageDTO(language.getCode(), language.getName()));
    }
    
    // ============ NATIONALITIES ============
    
    @GetMapping("/nationalities")
    public ResponseEntity<List<NationalityDTO>> getAllNationalities() {
        List<NationalityDTO> nationalities = nationalityRepository.findAll().stream()
                .map(n -> new NationalityDTO(n.getCode(), n.getName()))
                .collect(Collectors.toList());
        return ResponseEntity.ok(nationalities);
    }
    
    @GetMapping("/nationalities/{code}")
    public ResponseEntity<NationalityDTO> getNationality(@PathVariable String code) {
        return nationalityRepository.findById(code)
                .map(n -> ResponseEntity.ok(new NationalityDTO(n.getCode(), n.getName())))
                .orElseGet(() -> ResponseEntity.notFound().build());
    }
    
    @PostMapping("/nationalities")
    public ResponseEntity<NationalityDTO> createNationality(
            @RequestBody NationalityDTO nationalityDTO) {
        var nationality = nationalityRepository.save(new Nationality(
                nationalityDTO.getCode(),
                nationalityDTO.getName()
        ));
        return ResponseEntity.status(HttpStatus.CREATED)
                .body(new NationalityDTO(nationality.getCode(), nationality.getName()));
    }
}
```

**Endpoints** :
- `GET /api/v1/reference/languages` : Toutes les langues
- `GET /api/v1/reference/languages/{code}` : Détail langue
- `POST /api/v1/reference/languages` : Ajouter langue
- `GET /api/v1/reference/nationalities` : Toutes nationalités
- `GET /api/v1/reference/nationalities/{code}` : Détail nationalité
- `POST /api/v1/reference/nationalities` : Ajouter nationalité

## 🎯 Annotations Utilisées

### @RestController
```java
@RestController
public class MyController { }
// = @Controller + @ResponseBody automatique
```

### @RequestMapping
```java
@RequestMapping("/api/v1/books")
// Préfixe pour tous les endpoints
```

### @GetMapping, @PostMapping, etc.
```java
@GetMapping("/{id}")          // Equivalent à @RequestMapping(method=GET)
@PostMapping                  // Equivalent à @RequestMapping(method=POST)
@PutMapping("/{id}")          // PUT
@DeleteMapping("/{id}")       // DELETE
```

### @PathVariable
```java
@GetMapping("/{id}")
public void get(@PathVariable Long id) { }
// id vient du chemin URL
```

### @RequestBody
```java
@PostMapping
public void create(@RequestBody BookDTO bookDTO) { }
// Convertit JSON body en objet
```

### @Valid
```java
@PostMapping
public void create(@Valid @RequestBody BookDTO bookDTO) { }
// Déclenche validation Jakarta Validation
```

### @RequestParam
```java
@GetMapping("/search/title")
public void search(@RequestParam String title) { }
// Récupère query parameter (?title=query)
```

### ResponseEntity
```java
// Contrôle complet de la réponse HTTP
ResponseEntity.status(HttpStatus.CREATED).body(data)
ResponseEntity.ok(data)
ResponseEntity.noContent().build()
ResponseEntity.notFound().build()
```

## 🔄 Cycle de Vie d'une Requête

```
1. Client envoie HTTP Request
   ↓
2. @RequestMapping mappe à l'endpoint
   ↓
3. @PathVariable/@RequestParam extraient params
   ↓
4. @RequestBody désérialise JSON
   ↓
5. @Valid valide le DTO (Jakarta Validation)
   ↓
6. Service est appelé
   ↓
7. Service retourne DTO
   ↓
8. Controller retourne ResponseEntity
   ↓
9. DTO est sérialisé en JSON
   ↓
10. HTTP Response envoyée au client
```
