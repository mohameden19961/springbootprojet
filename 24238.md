### BookRepository
```java
@Repository
public interface BookRepository extends JpaRepository<Book, Long> {
    
    @Query("SELECT b FROM Book b WHERE b.deleted = false ORDER BY b.title")
    Page<Book> findAllActive(Pageable pageable);
    
    @Query("SELECT b FROM Book b WHERE b.deleted = false AND b.isbn = :isbn")
    Book findByIsbnActive(String isbn);
    
    @Query("SELECT b FROM Book b WHERE b.deleted = false AND b.title LIKE %:title%")
    Page<Book> findByTitleContainingActive(String title, Pageable pageable);
    
    // ✅ Recherche avec JOIN sur auteurs
    @Query("SELECT b FROM Book b JOIN b.authors a WHERE b.deleted = false AND a.id = :authorId")
    Page<Book> findByAuthorIdActive(Long authorId, Pageable pageable);
    
    @Query("SELECT b FROM Book b WHERE b.deleted = false AND b.category.id = :categoryId")
    Page<Book> findByCategoryIdActive(Long categoryId, Pageable pageable);
    
    @Query("SELECT b FROM Book b WHERE b.deleted = false AND b.language.code = :languageCode")
    Page<Book> findByLanguageCodeActive(String languageCode, Pageable pageable);
}
```

**Requêtes Clés** :
- `findAllActive()` : Tous les livres
- `findByIsbnActive()` : Recherche par ISBN unique
- `findByTitleContainingActive()` : Recherche LIKE sur titre
- `findByAuthorIdActive()` : JOIN avec Authors pour trouver livres d'un auteur
- `findByCategoryIdActive()` : Livres d'une catégorie
- `findByLanguageCodeActive()` : Livres dans une langue

### ReservationRepository
```java
@Repository
public interface ReservationRepository extends JpaRepository<Reservation, Long> {
    
    @Query("SELECT r FROM Reservation r WHERE r.deleted = false AND r.member.id = :memberId ORDER BY r.createdAt DESC")
    Page<Reservation> findByMemberIdActive(Long memberId, Pageable pageable);
    
    @Query("SELECT r FROM Reservation r WHERE r.deleted = false AND r.book.id = :bookId ORDER BY r.queuePosition ASC")
    Page<Reservation> findByBookIdActive(Long bookId, Pageable pageable);
    
    // ✅ Vérifier si un membre a déjà réservé ce livre
    @Query("SELECT r FROM Reservation r WHERE r.deleted = false AND r.member.id = :memberId AND r.book.id = :bookId AND r.status = 'PENDING'")
    Optional<Reservation> findPendingReservation(Long memberId, Long bookId);
    
    // ✅ TRÈS IMPORTANT : Obtenir premier réservant (file d'attente)
    @Query("SELECT r FROM Reservation r WHERE r.deleted = false AND r.book.id = :bookId AND r.status = 'PENDING' ORDER BY r.queuePosition ASC LIMIT 1")
    Optional<Reservation> findFirstPendingReservation(Long bookId);
    
    @Query("SELECT r FROM Reservation r WHERE r.deleted = false AND r.status = :status ORDER BY r.createdAt DESC")
    Page<Reservation> findByStatusActive(ReservationStatus status, Pageable pageable);
}
```

**Requêtes Critiques** :
- `findPendingReservation()` : Vérifier si déjà réservé
- `findFirstPendingReservation()` : **PREMIÈRE PERSONNE EN FILE** (file FIFO)

## 🛠️ Services Créés

### CategoryService
```java
@Service
@RequiredArgsConstructor
@Transactional
public class CategoryService {
    
    private final CategoryRepository categoryRepository;
    
    // CREATE
    public CategoryDTO createCategory(CategoryDTO categoryDTO) {
        // ✅ Vérifier unicité du nom
        if (categoryRepository.findByNameActive(categoryDTO.getName()) != null) {
            throw new DuplicateResourceException("Catégorie existe déjà");
        }
        // ✅ Créer et sauvegarder
        Category category = Category.builder()
                .name(categoryDTO.getName())
                .deleted(false)
                .build();
        return mapToDTO(categoryRepository.save(category));
    }
    
    // READ
    @Transactional(readOnly = true)
    public Page<CategoryDTO> getAllCategories(Pageable pageable) {
        return categoryRepository.findAllActive(pageable).map(this::mapToDTO);
    }
    
    @Transactional(readOnly = true)
    public CategoryDTO getCategoryById(Long id) {
        Category category = categoryRepository.findById(id)
                .filter(c -> !c.getDeleted())  // Vérifier soft delete
                .orElseThrow(() -> new ResourceNotFoundException("Catégorie non trouvée"));
        return mapToDTO(category);
    }
    
    // UPDATE
    public CategoryDTO updateCategory(Long id, CategoryDTO categoryDTO) {
        Category category = categoryRepository.findById(id)
                .filter(c -> !c.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Catégorie non trouvée"));
        
        // Vérifier unicité si modifié
        if (!category.getName().equals(categoryDTO.getName()) && 
            categoryRepository.findByNameActive(categoryDTO.getName()) != null) {
            throw new DuplicateResourceException("Nom existe déjà");
        }
        
        category.setName(categoryDTO.getName());
        return mapToDTO(categoryRepository.save(category));
    }
    
    // DELETE (soft delete)
    public void deleteCategory(Long id) {
        Category category = categoryRepository.findById(id)
                .filter(c -> !c.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Catégorie non trouvée"));
        
        category.setDeleted(true);
        categoryRepository.save(category);
    }
    
    // Mapper
    private CategoryDTO mapToDTO(Category category) {
        return CategoryDTO.builder()
                .id(category.getId())
                .name(category.getName())
                .build();
    }
}
```

**Pattern CRUD** :
- CREATE : Créer + valider unicité
- READ : Récupérer + vérifier soft delete
- UPDATE : Modifier + re-valider unicité
- DELETE : Soft delete (deleted=true)

### AuthorService
```java
@Service
@RequiredArgsConstructor
@Transactional
public class AuthorService {
    
    private final AuthorRepository authorRepository;
    private final NationalityRepository nationalityRepository;
    
    public AuthorDTO createAuthor(AuthorDTO authorDTO) {
        // Récupérer la nationalité si fournie
        Nationality nationality = null;
        if (authorDTO.getNationalityCode() != null) {
            nationality = nationalityRepository.findById(authorDTO.getNationalityCode())
                    .orElseThrow(() -> new ResourceNotFoundException("Nationalité non trouvée"));
        }
        
        Author author = Author.builder()
                .name(authorDTO.getName())
                .nationality(nationality)
                .deleted(false)
                .build();
        
        return mapToDTO(authorRepository.save(author));
    }
    
    // Recherche avec LIKE
    @Transactional(readOnly = true)
    public Page<AuthorDTO> searchAuthors(String name, Pageable pageable) {
        return authorRepository.findByNameContainingActive(name, pageable)
                .map(this::mapToDTO);
    }
    
    // ... (CRUD standard)
    
    private AuthorDTO mapToDTO(Author author) {
        return AuthorDTO.builder()
                .id(author.getId())
                .name(author.getName())
                .nationalityCode(author.getNationality() != null ? author.getNationality().getCode() : null)
                .nationality(author.getNationality() != null ? 
                        new NationalityDTO(author.getNationality().getCode(), author.getNationality().getName()) : null)
                .build();
    }
}
```

**Particularités** :
- Gère la relation vers Nationality
- Recherche par nom avec LIKE

### BookService
```java
@Service
@RequiredArgsConstructor
@Transactional
public class BookService {
    
    private final BookRepository bookRepository;
    private final BookItemRepository bookItemRepository;
    private final LanguageRepository languageRepository;
    private final CategoryRepository categoryRepository;
    private final PublisherRepository publisherRepository;
    private final AuthorRepository authorRepository;
    
    // ============ BOOK OPERATIONS ============
    
    public BookDTO createBook(BookDTO bookDTO) {
        // Vérifier ISBN unique
        if (bookRepository.findByIsbnActive(bookDTO.getIsbn()) != null) {
            throw new DuplicateResourceException("ISBN existe déjà");
        }
        
        // Récupérer et valider dépendances
        Language language = languageRepository.findById(bookDTO.getLanguageCode())
                .orElseThrow(() -> new ResourceNotFoundException("Langue non trouvée"));
        
        Category category = categoryRepository.findById(bookDTO.getCategoryId())
                .filter(c -> !c.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Catégorie non trouvée"));
        
        Publisher publisher = publisherRepository.findById(bookDTO.getPublisherId())
                .filter(p -> !p.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Éditeur non trouvé"));
        
        // Créer le livre
        Book book = Book.builder()
                .title(bookDTO.getTitle())
                .isbn(bookDTO.getIsbn())
                .language(language)
                .category(category)
                .publisher(publisher)
                .deleted(false)
                .build();
        
        return mapToDTO(bookRepository.save(book));
    }
    
    // Recherches multiples
    @Transactional(readOnly = true)
    public Page<BookDTO> searchBooksByTitle(String title, Pageable pageable) {
        return bookRepository.findByTitleContainingActive(title, pageable)
                .map(this::mapToDTO);
    }
    
    @Transactional(readOnly = true)
    public Page<BookDTO> findBooksByCategory(Long categoryId, Pageable pageable) {
        // Vérifier catégorie existe
        categoryRepository.findById(categoryId)
                .filter(c -> !c.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Catégorie non trouvée"));
        
        return bookRepository.findByCategoryIdActive(categoryId, pageable)
                .map(this::mapToDTO);
    }
    
    @Transactional(readOnly = true)
    public Page<BookDTO> findBooksByAuthor(Long authorId, Pageable pageable) {
        // Vérifier auteur existe
        authorRepository.findById(authorId)
                .filter(a -> !a.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Auteur non trouvé"));
        
        return bookRepository.findByAuthorIdActive(authorId, pageable)
                .map(this::mapToDTO);
    }
    
    // ============ BOOK ITEM OPERATIONS ============
    
    public BookItemDTO createBookItem(Long bookId, BookItemDTO bookItemDTO) {
        // Vérifier le livre existe
        Book book = bookRepository.findById(bookId)
                .filter(b -> !b.getDeleted())
                .orElseThrow(() -> new ResourceNotFoundException("Livre non trouvé"));
        
        // Vérifier barcode unique
        if (bookItemRepository.findByBarcodeActive(bookItemDTO.getBarcode()) != null) {
            throw new DuplicateResourceException("Barcode existe déjà");
        }
        
        // Créer l'exemplaire
        BookItem bookItem = BookItem.builder()
                .barcode(bookItemDTO.getBarcode())
                .status(BookItemStatus.AVAILABLE)
                .book(book)
                .deleted(false)
                .build();
        
        return mapBookItemToDTO(bookItemRepository.save(bookItem));
    }
    
    @Transactional(readOnly = true)
    public long countAvailableItemsByBook(Long bookId) {
        return bookItemRepository.countAvailableByBookId(bookId);
    }
}
```

**Complexité** :
- Gère plusieurs dépendances (Language, Category, Publisher, Author)
- Créer et lister des BookItems

## 🎯 Patterns et Bonnes Pratiques

### 1. Transactional Management
```java
@Service
@Transactional  // Pour writes
public class MyService {
    
    public void updateData() { }  // Transactional par défaut
    
    @Transactional(readOnly = true)  // Pour reads
    public Page<Data> getData(Pageable p) { }
}
```

### 2. Vérification Soft Delete
```java
// ❌ Mauvais
entity = repository.findById(id);

// ✅ Bon
entity = repository.findById(id)
    .filter(e -> !e.getDeleted())
    .orElseThrow(...);
```

### 3. Validation Unicité
```java
// ❌ Mauvais (pas de vérification)
repository.save(entity);

// ✅ Bon
if (repository.findByFieldActive(value) != null) {
    throw new DuplicateResourceException(...);
}
```

### 4. Pagination
```java
// ✅ Retourner Page au lieu de List
Page<EntityDTO> results = repository.findAllActive(pageable)
    .map(this::mapToDTO);
```

### 5. RequiredArgsConstructor (Lombok)
```java
@RequiredArgsConstructor  // Injecte toutes les dependencies
public class MyService {
    private final Repository repo;
}
```
