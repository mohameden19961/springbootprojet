## 2. ENTITÉS JPA

### Book (`@SQLRestriction("deleted = false")`)
- `id` (auto-généré), `title`, `isbn` (unique)
- `language` → `@ManyToOne` (LAZY)
- `category` → `@ManyToOne` (LAZY, `@NotFound(IGNORE)`)
- `publisher` → `@ManyToOne` (LAZY, `@NotFound(IGNORE)`)
- `bookItems` → `@OneToMany(cascade = ALL, orphanRemoval = true)`
- Lombok : `@Builder`, `@Getter`, `@Setter`, `@NoArgsConstructor`, `@AllArgsConstructor`

### BookItem (`@Version` pour Optimistic Locking)
- `id`, `barcode` (unique), `status` (AVAILABLE par défaut)
- `version` (`@Version` Long) — verrouillage optimiste
- `book` → `@ManyToOne` (LAZY, `@JsonIgnore`)
- Lombok : `@Builder`, `@Getter`, `@Setter`, `@NoArgsConstructor`, `@AllArgsConstructor`

### BookAuthor (clé composite avec `@EmbeddedId`)
- `id` → `BookAuthorId(bookId, authorId)` — `@Embeddable`
- `book` + `author` → `@ManyToOne` avec `@MapsId`
- `role` → `AuthorRole` (MAIN_AUTHOR, CO_AUTHOR, EDITOR)
- `BookAuthorId` : `@EqualsAndHashCode`, `@AllArgsConstructor`, `@NoArgsConstructor`
- Lombok : `@Builder`, `@Getter`, `@Setter`, `@NoArgsConstructor`, `@AllArgsConstructor`

---

## 3. REPOSITORIES

- **BookRepository** : `findByIsbn(String)` → `Optional<Book>`
- **BookItemRepository** : `findByBarcode(String)` → `Optional<BookItem>`
- **BookAuthorRepository** : `findByBookId(@Param("bookId") Long)` → `List<BookAuthor>` (via `@Query`)

---

## 4. DTOs

### Requête
- **BookDTO** : `title`, `isbn`, `languageCode`, `categoryId`, `publisherId` (avec validation)
- **BookItemDTO** : `barcode`, `status` (optionnel)

### Réponse
- **BookResponse** : `id`, `title`, `isbn`, `languageCode`, `languageName`, `categoryId`, `categoryName`, `publisherId`, `publisherName`
- **BookItemResponse** : `id`, `barcode`, `status`, `version`, `bookId`
- **BookAuthorResponse** : `bookId`, `bookTitle`, `authorId`, `authorName`, `role`

---

## 5. SERVICES

### BookService
- `findAll(Pageable)` → `Page<BookResponse>`
- `findById(Long)` → `BookResponse`
- `create(BookDTO)` → `BookResponse` (vérifie ISBN unique)
- `update(Long, BookDTO)` → `BookResponse` (vérifie ISBN unique)
- `delete(Long)` → soft-delete

### BookItemService
- `findAllByBook(Long)` → `List<BookItemResponse>`
- `findById(Long)` → `BookItemResponse`
- `create(Long, BookItemDTO)` → `BookItemResponse` (vérifie code-barres unique)
- `update(Long, BookItemDTO)` → `BookItemResponse`
- `delete(Long)` → soft-delete

### BookAuthorService
- `findAllByBook(Long)` → `List<BookAuthorResponse>`
- `addAuthorToBook(Long, Long, AuthorRole)` → `BookAuthorResponse`
- `removeAuthorFromBook(Long, Long)` → void

Tous les services utilisent des exceptions typées :
- `ResourceNotFoundException` (404)
- `DuplicateResourceException` (409)

## 7. Optimistic Locking

- `BookItem` contient un champ `@Version private Long version`
- Hibernate incrémente automatiquement la version à chaque update
- En cas de concurence, `OptimisticLockException` est levée
- La version est retournée dans `BookItemResponse` pour le suivi client
