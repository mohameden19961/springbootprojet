## 🎯 BorrowService - Service Critique

### Concept Principal

Un **Borrow** (emprunt) représente l'action de prendre un livre :
```
Member → emprunte → BookItem

Exemplaire passe de AVAILABLE à BORROWED
Borrow crée une trace historique
Duration = 30 jours
Max renouvellements = 3
```

---

### 1. Emprunter un Livre (borrowBook)

#### Logique Métier Complexe

```java
public BorrowDTO borrowBook(Long memberId, Long bookItemId) {
    
    // ÉTAPE 1 : Valider les données existent
    Member member = memberRepository.findById(memberId)
            .filter(m -> !m.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    BookItem bookItem = bookItemRepository.findById(bookItemId)
            .filter(bi -> !bi.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    // ÉTAPE 2 : Vérifier que l'exemplaire est AVAILABLE
    // ❌ ERREUR : Ne pas emprunter si BORROWED, RESERVED, LOST, DAMAGED
    if (bookItem.getStatus() != BookItemStatus.AVAILABLE) {
        throw new BusinessException(
            "Book item is not available for borrowing. Current status: " + 
            bookItem.getStatus()
        );
    }
    
    // ÉTAPE 3 : Vérifier limite d'emprunts du membre
    // ❌ ERREUR : Dépasser maxBorrows
    long activeBorrows = borrowRepository.countActiveBorrowsByMemberId(memberId);
    if (activeBorrows >= member.getMaxBorrows()) {
        throw new BusinessException(
            "Member has reached maximum number of active borrows: " + 
            member.getMaxBorrows()
        );
    }
    
    // ÉTAPE 4 : Vérifier réservation prioritaire
    // ❌ ERREUR : Un autre membre a réservé ce livre
    var firstReservation = reservationRepository
            .findFirstPendingReservation(bookItem.getBook().getId());
    
    if (firstReservation.isPresent() && 
        !firstReservation.get().getMember().getId().equals(memberId)) {
        throw new BusinessException(
            "Book has a priority reservation from another member"
        );
    }
    
    // ÉTAPE 5 : Créer l'emprunt (avec optimistic locking via version)
    Borrow borrow = Borrow.builder()
            .member(member)
            .bookItem(bookItem)
            .status(BorrowStatus.ACTIVE)
            .renewalCount(0)
            .borrowedAt(LocalDateTime.now())
            .dueDate(LocalDateTime.now().plusDays(BORROW_DURATION_DAYS))  // 30j
            .deleted(false)
            .build();
    
    // ÉTAPE 6 : Mettre à jour le statut du BookItem
    // ⚠️ OPTIMISTIC LOCK : Si 2 utilisateurs simultanément → Exception
    bookItem.setStatus(BookItemStatus.BORROWED);
    bookItemRepository.save(bookItem);  // Ceci incrémente version
    
    // ÉTAPE 7 : Annuler réservation si présente
    if (firstReservation.isPresent()) {
        var reservation = firstReservation.get();
        reservation.setStatus(ReservationStatus.COMPLETED);
        reservationRepository.save(reservation);
    }
    
    // ÉTAPE 8 : Sauvegarder l'emprunt
    Borrow saved = borrowRepository.save(borrow);
    
    return mapToDTO(saved);
}
```

### 2. Retourner un Livre (returnBook)

```java
public BorrowDTO returnBook(Long borrowId) {
    
    // ÉTAPE 1 : Vérifier emprunt existe
    Borrow borrow = borrowRepository.findById(borrowId)
            .filter(b -> !b.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    // ÉTAPE 2 : Vérifier que l'emprunt est retournable
    // ❌ ERREUR : Ne pas retourner si déjà RETURNED
    if (borrow.getStatus() != BorrowStatus.ACTIVE && 
        borrow.getStatus() != BorrowStatus.OVERDUE) {
        throw new BusinessException(
            "Cannot return book with status: " + borrow.getStatus()
        );
    }
    
    BookItem bookItem = borrow.getBookItem();
    
    // ÉTAPE 3 : Mettre à jour l'emprunt
    borrow.setStatus(BorrowStatus.RETURNED);
    borrow.setReturnedAt(LocalDateTime.now());
    
    // ÉTAPE 4 : Vérifier s'il y a une réservation en attente
    var firstReservation = reservationRepository
            .findFirstPendingReservation(bookItem.getBook().getId());
    
    if (firstReservation.isPresent()) {
        // ✅ CAS 1 : Il y a un réservant → Livre devient RESERVED
        bookItem.setStatus(BookItemStatus.RESERVED);
        
        var reservation = firstReservation.get();
        reservation.setStatus(ReservationStatus.READY);  // Notification
        reservationRepository.save(reservation);
    } else {
        // ✅ CAS 2 : Pas de réservant → Livre redevient AVAILABLE
        bookItem.setStatus(BookItemStatus.AVAILABLE);
    }
    
    // ÉTAPE 5 : Sauvegarder les changements
    bookItemRepository.save(bookItem);
    Borrow updated = borrowRepository.save(borrow);
    
    return mapToDTO(updated);
}
```

### 3. Renouveler un Emprunt (renewBorrow)

```java
public BorrowDTO renewBorrow(Long borrowId) {
    
    // ÉTAPE 1 : Vérifier emprunt
    Borrow borrow = borrowRepository.findById(borrowId)
            .filter(b -> !b.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    // ÉTAPE 2 : Vérifier que l'emprunt est ACTIVE
    if (borrow.getStatus() != BorrowStatus.ACTIVE) {
        throw new BusinessException(
            "Cannot renew borrow with status: " + borrow.getStatus()
        );
    }
    
    // ÉTAPE 3 : Vérifier nombre de renouvellements
    // ❌ ERREUR : Max 3 renouvellements par emprunt
    if (borrow.getRenewalCount() >= MAX_RENEWAL_COUNT) {  // MAX = 3
        throw new BusinessException(
            "Maximum renewal count (" + MAX_RENEWAL_COUNT + ") reached"
        );
    }
    
    // ÉTAPE 4 : Vérifier s'il existe une réservation
    // ❌ ERREUR : Ne pas renouveler si quelqu'un réserve
    var existingReservation = reservationRepository
            .findFirstPendingReservation(borrow.getBookItem().getBook().getId());
    
    if (existingReservation.isPresent()) {
        throw new BusinessException(
            "Cannot renew borrow as there is an active reservation"
        );
    }
    
    // ÉTAPE 5 : Renouveler
    borrow.setRenewalCount(borrow.getRenewalCount() + 1);
    borrow.setDueDate(LocalDateTime.now().plusDays(BORROW_DURATION_DAYS));  // +30j
    
    // ÉTAPE 6 : Sauvegarder
    Borrow updated = borrowRepository.save(borrow);
    
    return mapToDTO(updated);
}
```

## 🔔 ReservationService - Gestion de la File d'Attente

### Concept Principal

Une **Réservation** permet de réserver un livre quand aucun exemplaire n'est disponible :

```
Aucun exemplaire AVAILABLE
    ↓
Member réserve le livre
    ↓
Automatiquement mis en file d'attente
    ↓
Quand exemplaire retourné :
    → Premier réservant obtient notification
```

---

### 1. Créer Réservation (createReservation)

```java
public ReservationDTO createReservation(Long memberId, Long bookId) {
    
    // ÉTAPE 1 : Valider données
    Member member = memberRepository.findById(memberId)
            .filter(m -> !m.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    Book book = bookRepository.findById(bookId)
            .filter(b -> !b.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    // ÉTAPE 2 : Vérifier qu'AUCUN exemplaire n'est disponible
    // ❌ ERREUR : Impossible réserver si exemplaires dispo
    if (bookItemRepository.countAvailableByBookId(bookId) > 0) {
        throw new BusinessException(
            "Cannot create reservation: book has available copies"
        );
    }
    
    // ÉTAPE 3 : Vérifier que le membre n'a pas déjà réservé ce livre
    // ❌ ERREUR : Réservation en double
    var existingReservation = reservationRepository
            .findPendingReservation(memberId, bookId);
    
    if (existingReservation.isPresent()) {
        throw new BusinessException(
            "Member already has an active reservation for this book"
        );
    }
    
    // ÉTAPE 4 : Déterminer la position en file d'attente (FIFO)
    // ✅ IMPORTANT : Calculer automatiquement la position
    var lastReservation = reservationRepository
            .findByBookIdActive(bookId, Pageable.ofSize(1))
            .stream()
            .findFirst();
    
    Integer queuePosition = lastReservation.isPresent() ? 
            lastReservation.get().getQueuePosition() + 1 : 1;
    
    // ÉTAPE 5 : Créer la réservation
    Reservation reservation = Reservation.builder()
            .member(member)
            .book(book)
            .queuePosition(queuePosition)
            .status(ReservationStatus.PENDING)
            .createdAt(LocalDateTime.now())
            .deleted(false)
            .build();
    
    // ÉTAPE 6 : Sauvegarder
    Reservation saved = reservationRepository.save(reservation);
    
    return mapToDTO(saved);
}
```

#### Exemple de File d'Attente

### 2. Annuler Réservation (cancelReservation)

```java
public void cancelReservation(Long id) {
    
    // ÉTAPE 1 : Récupérer
    Reservation reservation = reservationRepository.findById(id)
            .filter(r -> !r.getDeleted())
            .orElseThrow(() -> new ResourceNotFoundException(...));
    
    // ÉTAPE 2 : Vérifier que la réservation peut être annulée
    if (reservation.getStatus() == ReservationStatus.COMPLETED || 
        reservation.getStatus() == ReservationStatus.CANCELLED) {
        throw new BusinessException(
            "Cannot cancel reservation with status: " + reservation.getStatus()
        );
    }
    
    // ÉTAPE 3 : Annuler
    reservation.setStatus(ReservationStatus.CANCELLED);
    
    // ÉTAPE 4 : Réorganiser la file d'attente
    // ✅ IMPORTANT : Recalculer les positions
    reorganizeQueue(reservation.getBook().getId());
    
    // ÉTAPE 5 : Sauvegarder
    reservationRepository.save(reservation);
}

@Transactional
public void reorganizeQueue(Long bookId) {
    // Récupérer toutes les réservations PENDING du livre
    var reservations = reservationRepository
            .findByBookIdActive(bookId, Pageable.ofSize(1000))
            .getContent();
    
    // Recalculer positions (1, 2, 3, ...)
    int position = 1;
    for (Reservation reservation : reservations) {
        if (reservation.getStatus() == ReservationStatus.PENDING) {
            reservation.setQueuePosition(position);
            position++;
            reservationRepository.save(reservation);
        }
    }
}
```

## 🛡️ Gestion des Exceptions Centralisée

### GlobalExceptionHandler

```java
@RestControllerAdvice
public class GlobalExceptionHandler {
    
    // 404 - Ressource non trouvée
    @ExceptionHandler(ResourceNotFoundException.class)
    public ResponseEntity<ErrorResponse> handleResourceNotFound(
            ResourceNotFoundException ex, 
            WebRequest request) {
        ErrorResponse error = ErrorResponse.builder()
                .timestamp(LocalDateTime.now())
                .status(HttpStatus.NOT_FOUND.value())
                .error("Resource Not Found")
                .message(ex.getMessage())
                .path(request.getDescription(false).replace("uri=", ""))
                .build();
        return new ResponseEntity<>(error, HttpStatus.NOT_FOUND);
    }
    
    // 409 - Ressource dupliquée
    @ExceptionHandler(DuplicateResourceException.class)
    public ResponseEntity<ErrorResponse> handleDuplicateResource(
            DuplicateResourceException ex, 
            WebRequest request) {
        ErrorResponse error = ErrorResponse.builder()
                .timestamp(LocalDateTime.now())
                .status(HttpStatus.CONFLICT.value())
                .error("Duplicate Resource")
                .message(ex.getMessage())
                .path(request.getDescription(false).replace("uri=", ""))
                .build();
        return new ResponseEntity<>(error, HttpStatus.CONFLICT);
    }
    
    // 400 - Violation de règle métier
    @ExceptionHandler(BusinessException.class)
    public ResponseEntity<ErrorResponse> handleBusinessException(
            BusinessException ex, 
            WebRequest request) {
        ErrorResponse error = ErrorResponse.builder()
                .timestamp(LocalDateTime.now())
                .status(HttpStatus.BAD_REQUEST.value())
                .error("Business Rule Violation")
                .message(ex.getMessage())
                .path(request.getDescription(false).replace("uri=", ""))
                .build();
        return new ResponseEntity<>(error, HttpStatus.BAD_REQUEST);
    }
    
    // 400 - Erreur de validation
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ResponseEntity<ErrorResponse> handleValidationError(
            MethodArgumentNotValidException ex, 
            WebRequest request) {
        Map<String, String> fieldErrors = new HashMap<>();
        
        ex.getBindingResult().getFieldErrors().forEach(error ->
                fieldErrors.put(error.getField(), error.getDefaultMessage())
        );
        
        ErrorResponse error = ErrorResponse.builder()
                .timestamp(LocalDateTime.now())
                .status(HttpStatus.BAD_REQUEST.value())
                .error("Validation Error")
                .message("Invalid input parameters")
                .validationErrors(fieldErrors)
                .path(request.getDescription(false).replace("uri=", ""))
                .build();
        return new ResponseEntity<>(error, HttpStatus.BAD_REQUEST);
    }
    
    // 500 - Erreur serveur
    @ExceptionHandler(Exception.class)
    public ResponseEntity<ErrorResponse> handleGlobalException(
            Exception ex, 
            WebRequest request) {
        ErrorResponse error = ErrorResponse.builder()
                .timestamp(LocalDateTime.now())
                .status(HttpStatus.INTERNAL_SERVER_ERROR.value())
                .error("Internal Server Error")
                .message(ex.getMessage())
                .path(request.getDescription(false).replace("uri=", ""))
                .build();
        return new ResponseEntity<>(error, HttpStatus.INTERNAL_SERVER_ERROR);
    }
}
```
